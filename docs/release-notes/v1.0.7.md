# 연금나침반 v1.0.7

삼성생명 IRP(개인형 퇴직연금) 계좌의 펀드/원리금보장형 비중 조정을 AI(Claude·Gemini·GPT)의 도움으로 계획하기 위한 Windows 데스크톱 도구입니다.

> ⚠️ **본 앱은 삼성생명보험주식회사와 무관한 비공식(third-party) 도구이며, AI가 생성한 제안은 참고용 정보일 뿐 투자 권유가 아닙니다.** 모든 투자 결정과 그 결과에 대한 책임은 사용자 본인에게 있습니다.

## v1.0.7 변경 사항

### 새 기능: 이력 피드백 루프 — 직전 추천의 실제 성과를 AI에게 같이 보여주기

v1.0.5에서 직전 회차의 *추천 텍스트*를 AI에게 참고용으로 첨부하는 기능을 도입했는데, 이번 회차에선 한 걸음 더 나아갑니다 — **그 추천 이후 실제로 잔고가 어떻게 움직였는지의 사실 데이터까지 AI가 함께 보고 자기 평가**할 수 있게 했습니다. AI가 자기 트랙 레코드를 인지한 상태에서 새 추천을 작성하므로 일관성이 올라가고, 결과가 좋지 않았던 방향은 자가 보정 가능성이 생깁니다.

- **AI 리밸런싱 화면 — 직전 회차 선택 시 실제 성과 배너 자동 표시**
  - 직전 회차 콤보에서 회차를 선택하면 즉시 콤보 아래에 light blue 카드가 나타남
  - 표시 내용: "직전 회차 → 현재: N일 동안 +X.XX%" (시작·현재 잔고, 신규 입금, 연환산까지)
  - 이 사실 데이터가 AI 프롬프트의 새 섹션 "## 직전 회차 → 현재 시점 실제 성과 (참고)"에 함께 실려 전달됨
  - AI에게는 "결과가 긍정적이고 시장 환경이 비슷하면 큰 폭 전환 자제 / 결과가 부정적이면 어디서 어긋났는지 짚어 새 추천에 반영 / 단기 결과 과대 해석 금지" 같은 자기 평가 가이드도 함께 전달

- **실현 수익률 계산은 결정론적 — 입금 차감 후 진짜 운용수익**
  - **DepositAmount 델타** 우선 사용 — 두 시점 모두 원금 정보가 있으면 "현재 원금 − 직전 원금"으로 신규 입금 정확히 차감
  - 원금 정보가 없으면 **월 적립액 × 경과 기간**으로 추정 (근사)
  - 둘 다 없으면 차감 안 한 gross 수익률로 표시하고 그 사실을 명시
  - **30일 미만 기간은 연환산 생략** — 짧은 기간을 연환산하면 노이즈가 폭발하므로

- **이력 화면 — 회차별 운용수익 한눈에 비교**
  - 좌측 회차 목록 각 행에 "다음 회차 대비 N일 — 운용수익 +X.XX%" 라벨이 자동 표시
  - 가장 최근 회차는 "현재 대비"로 표시 (현재 입력된 계좌 상태와 비교)
  - 어느 회차의 추천이 결과적으로 잘 작동했는지 시간순으로 한눈에 인지

### 보유 상품 변동 추적

직전 시점 보유 상품 vs 현재 보유 상품을 자동 비교해 **유지·매도·신규 매수**로 분류한 결과를 프롬프트에 같이 전달합니다. AI가 "사용자가 내 추천을 얼마나 따랐는지"를 추정해 새 추천 작성에 활용 가능.

## 설치 방법

다음 두 파일을 함께 받으세요:

| 파일 | 용도 |
| --- | --- |
| `PensionCompass_1.0.7.0_x64.msixbundle` | 앱 본체 |
| `PensionCompass_1.0.7.0_x64.cer` | self-signed 코드 서명 인증서 |

### 1단계: 인증서 신뢰 등록 (PC당 최초 1회)

자체 서명 인증서이므로 받는 PC가 발급자(CN=Changhwan Bang)를 신뢰하도록 등록해야 합니다.

1. `PensionCompass_1.0.7.0_x64.cer` 더블클릭
2. **인증서 설치(I)** 버튼 클릭
3. 저장소 위치: **로컬 컴퓨터(L)** 선택 → 다음 (UAC 동의 필요)
4. 인증서 저장소: **모든 인증서를 다음 저장소에 저장(P)** → 찾아보기 → **신뢰할 수 있는 루트 인증 기관** 선택 → 확인 → 다음 → 마침

> 자체 서명 인증서는 그 자체가 루트이기 때문에 "신뢰할 수 있는 게시자" 나 "신뢰할 수 있는 사람" 에 넣으면 체인 검증이 끊겨 `0x800B010A` (CERT_E_CHAINING) 오류로 설치가 실패합니다. 반드시 **로컬 컴퓨터 → 신뢰할 수 있는 루트 인증 기관** 조합이어야 합니다.

PowerShell 한 줄로도 가능 (관리자):

```pwsh
Import-Certificate -FilePath ".\PensionCompass_1.0.7.0_x64.cer" `
                   -CertStoreLocation Cert:\LocalMachine\Root
```

### 2단계: 앱 설치

`PensionCompass_1.0.7.0_x64.msixbundle` 더블클릭 → "설치" 버튼.

설치 완료 후 시작 메뉴에서 "연금나침반"으로 검색해 실행할 수 있습니다.

### 업데이트하는 경우

기존 버전이 이미 설치되어 있다면 새 .msixbundle을 더블클릭하면 in-place 업데이트됩니다 (인증서 재등록 불필요, 기존 입력 데이터 유지). 동기화 폴더 설정도 그대로 유지됩니다.

## 시스템 요구 사항

- Windows 10 17763(1809) 이상 / Windows 11
- x64 아키텍처
- AI 호출용 인터넷 연결 + 사용자 본인의 API Key (Anthropic Claude / Google Gemini / OpenAI 중 하나 이상)

## 데이터 보관

- **계좌·카탈로그·이력**: 사용자 PC의 앱 LocalState 폴더 + (지정 시) 동기화 폴더 양쪽에 미러 저장. JSON 평문이며 사용자 본인의 PC/클라우드 계정 안에서만 보입니다
- **API Key 3개**: Windows 자격증명 보관소 (사용자 계정으로 암호화). 동기화 폴더에 절대 기록되지 않음
- **AI 호출 시**: 보유 현황과 카탈로그가 사용자가 선택한 AI 공급자(Anthropic / Google / OpenAI)에 직접 전송됩니다. 그 외 외부 전송은 없습니다
