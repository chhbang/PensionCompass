# 연금나침반 v1.2.1

삼성생명 IRP(개인형 퇴직연금) 계좌의 펀드/원리금보장형 비중 조정을 AI(Claude·Gemini·GPT)의 도움으로 계획하기 위한 Windows 데스크톱 도구입니다.

> ⚠️ **본 앱은 삼성생명보험주식회사와 무관한 비공식(third-party) 도구이며, AI가 생성한 제안은 참고용 정보일 뿐 투자 권유가 아닙니다.** 모든 투자 결정과 그 결과에 대한 책임은 사용자 본인에게 있습니다.

## v1.2.1 변경 사항 — 게시자 표기 정리 (작은 패치)

설치된 앱 목록의 게시자(Publisher) 표기를 `turtulee` → **`Changhwan Bang`**으로 변경했습니다. 코드 서명 인증서의 발급자명(`CN=Changhwan Bang`)과 일관되게 맞추는 게 목적입니다.

- 기능적 변경은 없습니다.
- 인증서·OAuth 클라이언트·기타 식별자는 v1.2.0과 동일.
- v1.2.0이 이미 설치된 PC에서 새 .msixbundle을 더블클릭하면 in-place 업데이트되며 기존 데이터·동기 설정 그대로 유지됩니다.

v1.2.0 본 기능(Google 계정에 API Key 동기화 + 커버드콜 펀드 제외 옵션)은 [v1.2.0 릴리스 노트](v1.2.0.md)를 참조하세요.

## 설치 방법

[v1.2.0 설치 절차](v1.2.0.md)와 동일합니다. 인증서는 v1.2.0과 동일하므로 이미 신뢰 등록을 해두셨다면 .msixbundle만 더블클릭하시면 됩니다.
