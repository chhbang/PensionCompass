# 연금나침반 v1.0.4

삼성생명 IRP(개인형 퇴직연금) 계좌의 펀드/원리금보장형 비중 조정을 AI(Claude·Gemini·GPT)의 도움으로 계획하기 위한 Windows 데스크톱 도구입니다.

> ⚠️ **본 앱은 삼성생명보험주식회사와 무관한 비공식(third-party) 도구이며, AI가 생성한 제안은 참고용 정보일 뿐 투자 권유가 아닙니다.** 모든 투자 결정과 그 결과에 대한 책임은 사용자 본인에게 있습니다.

## v1.0.4 변경 사항

### 새 기능: 내 계좌 자동 입력

- 데이터 준비 화면에 **"내 계좌"** 행 추가. 삼성생명 ‘적립금/수익률 조회’ HTML을 그대로 불러오면 보유 상품과 합계가 자동 입력됩니다 — 7개 상품을 손으로 옮기는 부담이 사라집니다.
- 한 번 파싱한 결과를 CSV로도 보존할 수 있어, 같은 계좌 상태를 다시 불러올 때는 파싱 비용 없이 즉시 적용됩니다.
- "안내" 버튼을 누르면 `My삼성생명 → 퇴직연금 종합관리 → ‘총 적립금’ 클릭 → ‘다른 이름으로 저장’` 절차와 카탈로그 다운로드 절차가 함께 뜹니다.
- **개인정보 보호**: 파서는 가입자 성명·계좌번호·명세일자가 들어 있는 HTML 영역을 처음부터 읽지 않도록 설계되었습니다. 모델에 들어가는 값은 상품명·금액·수익률·운용일수·좌수뿐입니다. (테스트로도 검증)

### 보안: API Key를 Windows 자격증명 보관소로 이전

- 기존 빌드는 API Key를 LocalSettings에 평문으로 저장했지만, 이 버전부터는 **Windows 자격증명 보관소(`PasswordVault`)** 에 사용자 계정으로 암호화되어 보관됩니다. 디스크 탈취나 다른 사용자 계정에서의 접근은 차단됩니다.
- **자동 마이그레이션**: 새 빌드를 처음 실행할 때 기존 평문 API Key를 자동으로 보관소로 옮기고 LocalSettings에서 삭제합니다. 사용자가 따로 할 일은 없습니다.
- 한계: 같은 사용자 세션 안에서 실행되는 다른 프로그램이 동일 API로 꺼내 가는 것까지는 막지 못합니다. 공용 PC에서는 여전히 사용을 피해 주세요.

### 보유 상품 목록 UX (v1.0.3 이어서)

- 상품 중복 추가 차단 + 휴지통 버튼 위치/툴팁/헤더 정렬 정리는 v1.0.3 그대로 유지됩니다.

## 설치 방법

다음 두 파일을 함께 받으세요:

| 파일 | 용도 |
| --- | --- |
| `PensionCompass_1.0.4.0_x64.msixbundle` | 앱 본체 |
| `PensionCompass_1.0.4.0_x64.cer` | self-signed 코드 서명 인증서 |

### 1단계: 인증서 신뢰 등록 (PC당 최초 1회)

자체 서명 인증서이므로 받는 PC가 발급자(CN=Changhwan Bang)를 신뢰하도록 등록해야 합니다.

1. `PensionCompass_1.0.4.0_x64.cer` 더블클릭
2. **인증서 설치(I)** 버튼 클릭
3. 저장소 위치: **로컬 컴퓨터(L)** 선택 → 다음
4. 인증서 저장소: **모든 인증서를 다음 저장소에 저장(P)** → 찾아보기 → **신뢰할 수 있는 사람** 선택 → 확인 → 다음 → 마침

### 2단계: 앱 설치

`PensionCompass_1.0.4.0_x64.msixbundle` 더블클릭 → "설치" 버튼.

설치 완료 후 시작 메뉴에서 "연금나침반"으로 검색해 실행할 수 있습니다.

### 업데이트하는 경우

기존 버전이 이미 설치되어 있다면 새 .msixbundle을 더블클릭하면 in-place 업데이트됩니다 (인증서 재등록 불필요, 기존 입력 데이터 유지). API Key 보관 위치 마이그레이션은 첫 실행 시 자동으로 진행됩니다.

## 시스템 요구 사항

- Windows 10 17763(1809) 이상 / Windows 11
- x64 아키텍처
- AI 호출용 인터넷 연결 + 사용자 본인의 API Key (Anthropic Claude / Google Gemini / OpenAI 중 하나 이상)

## 데이터 보관

- **계좌 정보·상품 카탈로그**: 사용자 PC의 앱 LocalState 폴더 (`%LOCALAPPDATA%\Packages\<PFN>\LocalState\`)에 JSON으로 저장
- **API Key 3개**: Windows 자격증명 보관소 `PensionCompass.ApiKey`에 사용자 계정으로 암호화 보관
- **AI 호출 시**: 보유 현황과 카탈로그가 사용자가 선택한 AI 공급자(Anthropic / Google / OpenAI)에 직접 전송됩니다. 그 외 외부 전송은 없습니다.
